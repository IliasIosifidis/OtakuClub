# Otaku Club

Otaku Club is a modern Android application built with **Kotlin** and **Jetpack Compose**, focused on exploring anime content through a clean, responsive, and scalable UI.  
The project is designed as a **portfolio-grade application**, following current Android best practices and architectural patterns.

---

##  Features

- Browse anime content with a modern Compose-based UI
- Clean separation of concerns using **MVVM architecture**
- State-driven UI with unidirectional data flow
- Asynchronous image loading and smooth scrolling
- Modular, scalable codebase suitable for future expansion
- Production-ready project structure

---

##  Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose (Material 3)
- **Architecture:** MVVM
- **State Management:** ViewModel + State
- **Navigation:** Jetpack Navigation Compose
- **Async / Networking:** Coroutines
- **Image Loading:** Coil
- **Build System:** Gradle (Kotlin DSL)
